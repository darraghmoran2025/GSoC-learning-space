# Reviews

This folder contains reviews of other GSoC candidates' learning spaces and models.

Each review should live in its own file: `review_<github_username>.md`

---

## Review template

```markdown
# Review: <GitHub username>
**Repo:** <link>
**Date:** YYYY-MM-DD

## What they built
Brief description of the model(s).

## What I found interesting
What stood out — technically or conceptually.

## What I learned from reading their work
Something that changed how I think about Mesa or ABMs.

## Feedback I'd offer
Constructive notes if they asked.
```
